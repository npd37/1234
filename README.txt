# Anay Global — Static Website Preview

This is a static single-page website created to match the provided screenshots.

**How to use**
- Upload the contents of this folder to a GitHub repository.
- Enable GitHub Pages (Settings → Pages) with the `main` branch `/ (root)` to serve `index.html`.
- The site is fully static and requires no backend.

Files:
- index.html
- style.css
- script.js
- images/logo.png
